# Networking Lab 1 – Multi-LAN Routing & DHCP (Packet Tracer)

## Overview

This project was built using Cisco Packet Tracer to simulate a small enterprise network with two LANs connected through a router.

## Topology

* 2 Switches
* 1 Router
* 2 PCs (one per LAN)

## Key Configurations

* Configured router interfaces for two separate networks:

  * 192.168.1.0/24
  * 192.168.2.0/24
* Enabled inter-network routing through router interfaces
* Configured DHCP on router to automatically assign IP addresses
* Verified connectivity using ICMP (ping tests)

## Skills Demonstrated

* Network troubleshooting
* IP addressing and subnetting basics
* Router interface configuration
* DHCP configuration
* Basic network verification

## Results

* PCs successfully obtained IP addresses via DHCP
* End-to-end connectivity between LANs confirmed
