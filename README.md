Networking Lab 1 – Multi-LAN Routing and DHCP Configuration
Objective
Build a small network consisting of two separate LANs connected by a router and configure DHCP services to provide automatic IP address assignment. Verify communication between networks using connectivity testing tools.
Network Components
1 Router
2 Switches
2 PCs
Tasks Completed
Configured router interfaces for the 192.168.1.0/24 and 192.168.2.0/24 networks
Enabled routing between both LANs
Configured DHCP pools to automatically assign IP addresses to client devices
Verified correct IP addressing and default gateway assignment
Tested communication between hosts using ping
What I Learned
This lab helped reinforce how routers enable communication between different networks and how DHCP simplifies IP address management. I also gained experience verifying network connectivity and troubleshooting basic configuration issues within Packet Tracer.
Outcome
Both PCs successfully received IP addresses from DHCP and were able to communicate across separate LANs through the router.
